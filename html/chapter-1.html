Заголовки (Header)

В HTML существует шесть уровней заголовков.
Тег <h1> - основной заголовок.
Теги с <h2> по <h6> - подзаголовки.
Тег заголовков закрывающийся <h1></h1>.

Код в html:

<h1>Заголовок_1</h1>

<h2>Заголовок 2</h2>

<h3>Заголовок 3</h3>

<h4>Заголовок 4</h4>

<h5>Заголовок 5</h5>

<h6>Заголовок 6</h6>


Вывод кода:

Заголовок 1

Заголовок 2

Заголовок 3

Заголовок 4

Заголовок 5
Заголовок 6

Абзац (Paragraph)

Тег <p> применяется для формирования абзаца, по умолчанию каждый абзац отображается с новой строки. Тег абзаца закрывающийся <p></p>.

Код в html:

<p>"Текст"</p>

<p>Абзацы текста идущие друг за другом разделяются между собой отбивкой. Величиной отбивки можно управлять с помощью стилей. Если закрывающего тега нет, считается, что конец абзаца совпадает с началом следующего блочного элемента.</p>


Вывод кода:

"Текст"

Абзацы текста идущие друг за другом разделяются между собой отбивкой. Величиной отбивки можно управлять с помощью стилей. Если закрывающего тега нет, считается, что конец абзаца совпадает с началом следующего блочного элемента.


Полужирное и курсивное начертание (Bold and Italic)

Данные теги не имеют дополнительных значений к тексту.
Тег <b> используется для выделения ключевых слов.
Тег <i> отображает текст курсивом, для выделения технических терминов, иностранных слов и прочих включений в текст.
Данные теги требуют закрытия.

Код в html:

Lorem ipsum dolor sit, amet consectetur adipisicing elit. <b>Error, similique aperiam doloremque recusandae enim nesciunt. </b>Eaque quaerat laudantium porro ducimus tempora laboriosam quos voluptatum perspiciatis possimus non sequi numquam quod minus maiores, nulla molestiae excepturi eveniet provident incidunt distinctio odio. <i>Velit et enim, sint odio doloremque sequi ex consequuntur provident.</i>


Вывод кода:

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Error, similique aperiam doloremque recusandae enim nesciunt. Eaque quaerat laudantium porro ducimus tempora laboriosam quos voluptatum perspiciatis possimus non sequi numquam quod minus maiores, nulla molestiae excepturi eveniet provident incidunt distinctio odio. Velit et enim, sint odio doloremque sequi ex consequuntur provident.


Подстрочные и надстрочные индексы (Subscript and Superscript)

Тег <sup> используется для выделения символов которые должны быть отражены как надстрочные, к примеру, математические и физические формулы.
Тег <sub> используется дляотображения подстрочных символов, как правило используется в химических формулах.
Данные теги требуют закрытия.

Код в html:

Формула E=mc<sup>2</sup>. СО<sub>2</sub>


Вывод кода:

Формула E=mc2. CO2


Перенос строки и горизонтальная линия(Break line and Horizontal Rule)

Тег <br> применяется для перевода на новую строку в пределах одного абзаца, либо для формирования отступа.
Тег <hr> применяется для создания линии.
Данные теги не требуют закрытия.

Код в html:

Lorem ipsum dolor, <br> sit amet consectetur adipisicing elit. Natus, voluptatem. <hr> Lorem ipsum dolor sit amet consectetur, adipisicing elit. Aliquam, corporis.


Вывод кода:

Lorem ipsum dolor,
sit amet consectetur adipisicing elit. Natus, voluptatem.

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Aliquam, corporis.


Важная информация и акценты (Strong and Emphasis)

Тег <strong> используется для обозначения того, что текст помещенный в него имеет высокую степень важности. Браузеры по умолчанию отображают полужирным текстом как тег <b>.


Тег <em> сокращение от «emphasis» (акцент) используется для обозначения ударения. Программа, читающая текст, будет произносить слова, выделенные курсивом, с ударением, используя словесное ударение. Браузеры по умолчанию отображают курсивом как тег <i>.

Код в html:

<strong>Будьте осторожны:</strong> пельмешки <em>очень</em> горячие!


Вывод кода:

Будьте осторожны: пельмешки очень горячие!


Цитаты (Blockquote and Quote Element)

Тег <blockquote> используется для отображения длинных цитат внутри <blockquote> необходимо использовать <p>.
Браузеры склонны к выделению отступами.


Тег <q> сокращение от «quote» (цитата) используется для коротких цитат.
В обоих элементах может быть указан атрибут cite*.

Код в html:

<blockquote cite="http://www.vothouse.ru/films/ivan_vasiljevich_menjaet_professiju_quotes.html?ysclid=le2dhi0pid716794163">
<p> ―Ты прoсти, чтo я тебе мешаю, нo я дoлжна сooбщить тебе ужаснoе известие. У меня сегoдня в кафе увели перчатки. И я пoлюбила другoгo. Ты меня пoнимаешь, Шурик?</p>
</blockquote>

<p>
<q cite="https://citaty.info/character/vinni-puh"> Никто не может грустить, когда у него есть воздушный шарик!</q>
</p>

Вывод кода:

―Ты прoсти, чтo я тебе мешаю, нo я дoлжна сooбщить тебе ужаснoе известие. У меня сегoдня в кафе увели перчатки. И я пoлюбила другoгo. Ты меня пoнимаешь, Шурик?

Никто не может грустить, когда у него есть воздушный шарик!



Аббревиатура (Abbreviation)

Тег <abbr> указывает, что последовательность символов является аббревиатурой. С помощью атрибута title даётся расшифровка сокращения, что позволяет понимать аббревиатуру тем людям, которые с ней не знакомы. Кроме того, поисковые системы индексируют полнотекстовый вариант сокращения, это может использоваться для повышения рейтинга документа.

Код в html:

<abbr title="Abbreviation — аббревиатура. Аббревиатура - abbr">Abbr</abbr>

Вывод кода:

Abbr

Определение (Defintion)

Элемент <dfn> (от англ. definition — определение) применяется для выделения таких терминов при их первом появлении в тексте. Браузеры, как правило, отображают содержимое контейнера <dfn> с помощью курсивного начертания. Определение термина задаётся с помощью атрибута title у элемента <dfn>, либо с помощью атрибута title у вложенного элемента <abbr>, либо текстом внутри <dfn>.

Код в html:

<p><dfn>Капителью</dfn> в типографике называется текст, набранный прописными буквами уменьшенного размера.</p>

Вывод кода:

Капителью в типографике называется текст, набранный прописными буквами уменьшенного размера.
